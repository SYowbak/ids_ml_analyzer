# IDS ML Analyzer

IDS ML Analyzer (Intrusion Detection System Machine Learning Analyzer) — це комплексна система для виявлення мережевих аномалій та кібератак. Використовує машинне навчання (LightGBM, XGBoost, Isolation Forest) для аналізу мережевого трафіку (PCAP) та датасетів (CSV).

Проект оснащений інтуїтивним веб-інтерфейсом на базі Streamlit, системою автоматичної нормалізації даних (Unified Pipeline) та інтеграцією з AI (Google Gemini) для генерації детальних звітів та рекомендацій (SOC Analysis).

## 🚀 Ключові можливості

1. **Багатоформатний імпорт:** 
   - Нативна підтримка PCAP/PCAPNG файлів з автоматичною екстракцією ознак потоків (в стилі CIC-IDS).
   - Завантаження CSV файлів з гнучкою системою мапінгу колонок під єдину архітектуру (Unified Schema).
2. **Unified Data Pipeline (Expert Grade):**
   - Автоматичне вирівнювання даних між різними родинами датасетів (CIC-IDS2017/2018, NSL-KDD, UNSW-NB15).
   - Нормалізація одиниць виміру (байти/пакети/інтервали).
   - Фільтрація витоків даних (Data Leakage Filter).
3. **Advanced ML & Mega-Models:**
   - **Mega-Model (Multi-Class):** Єдина глобальна модель, здатна розпізнавати всі типи атак з різних сімейств.
   - **Two-Stage Architecture:** Двоетапний пайплайн (Детекція аномалій IsolationForest -> Класифікація LightGBM), що ефективно ловить Zero-Day атаки.
   - Оптимізація використання пам'яті: ліниве завантаження, batch-сканування (безпечно для >1M рядків) та stratified sampling.
4. **AI threat intelligence:**
   - Інтеграція з Google Gemini API для генерації зрозумілих людських пояснень і Executive Summaries для виявлених інцидентів та аномалій.
5. **Interactive UI & Dashboards:**
   - Багатий дашборд результатів: графіки (Threat Distribution), таблиці з підозрілими IP, експорт результатів у CSV та красиві PDF-звіти.

## ⚙️ Встановлення

Вимоги:
- Python 3.10 або новіший
- Рекомендується використовувати віртуальне середовище (`venv` або `uv`)

```bash
# Клонування репозиторію
git clone <repository_url>
cd IDS_ML_Analyzer

# Створення віртуального середовища
python -m venv .venv
# Активація (Windows)
.venv\Scripts\activate
# Активація (Linux/macOS)
source .venv/bin/activate

# Встановлення залежностей
pip install -r requirements.txt
```

## 🏁 Швидкий старт (Quick Start)

Запустіть застосунок через зручний batch-скрипт або напряму через Streamlit:

```bash
# Windows
run_app.bat

# Або напряму
streamlit run src/start_app.py
```

### Як користуватись системою:
1. Отримайте Google Gemini API ключ (опціонально) та введіть його в налаштуваннях для активації AI-підказок.
2. Перейдіть у вкладку **Навчання моделей**. Завантажте свій навчальний датасет (наприклад, CIC-IDS2017) та ініціюйте створення базової моделі.
3. Процес автоматично згенерує `preprocessor` та збереже архітектуру моделі.
4. Перейдіть до **Сканування (PCAP/CSV)**. Закиньте новий невідомий файл (лог або мережевий дамп).
5. Система проаналізує трафік, збереже контекст IP/Port та виведе розширений дашборд результатів, який можна завантажити у форматі PDF.

## 📂 Структура проекту
- `src/core/` - Основні класи для ML, нормалізації (Data Loader, Model Engine, Feature Aligner).
- `src/ui/` - Інтерфейс користувача на базі Streamlit, UI компоненти.
- `src/services/` - Каталог загроз, інтеграція зовнішніх API (Gemini).
- `models/` - Локальна директорія для збереження натренованих моделей (XGBoost/LightGBM, Joblib).
- `datasets/` - Місце для збереження тестових PCAP та CSV.
- `tests/` - Набір модульних та інтеграційних тестів для системи (PyTest).

## Розробка та Тестування
Проект використовує `pytest` для інтеграційного аудиту. Щоб запустити тести:
```bash
pytest tests/ -v
```

Забезпечується повна крос-сумісність між PCAP-сканером та CSV-тренінгом завдяки `FeatureAligner`.

## Ліцензія
Проект є Open-Source і призначений для підвищення мережевої безпеки та дослідження ML-рішень в у галузі IDS (Intrusion Detection Systems).
